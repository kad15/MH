Math utilities.
val miller_rabin_rounds : Z.t -> int
Return the number of rounds in the Miller-Rabin test to get an error probability <= 2^-80.

Reference: Handbook of applied cryptography 

val probab_prime : Z.t -> bool
probab_prime n determines whether n is probably prime or definitely composite by using the Miller-Rabin test.
val nextprime : Z.t -> Z.t
nextprime n returns the next probably prime number >= n.
val modinvert : Z.t -> Z.t -> Z.t
modinvert a m computes the modular multiplicative inverse of a modulo m.
Raises Failure if there is no solution
